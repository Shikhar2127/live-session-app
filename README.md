Live Session App
This is a small project where an Admin can start a live session and Students can join using a link.
The Admin plays a video, and students can watch the same video on their screen.

What this project does
Admin clicks “Start Session”
A unique join link is generated
Students open the link to join
A video player appears for both Admin and Students

Video controls include:
Play / Pause
Volume
Fullscreen
Playback speed
